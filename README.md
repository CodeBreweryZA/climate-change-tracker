# Climate Change Tracker

A single-page responsive landing page built with HTML, CSS, and JavaScript to raise awareness about climate change.

## Sections

- **Hero** – Bold introduction with call-to-action
- **About** – Overview of climate change and its impact
- **Features** – Key awareness features and statistics
- **Signup** – Newsletter / join-the-movement form
- **Blog** – Recent articles and news highlights
- **FAQ** – Frequently asked questions accordion
- **Footer** – Links, social icons, and credits

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling and interactive UI components
- Font Awesome icons and Google Fonts

## Getting Started

Open `index.html` in any modern browser. No build tools or server required.

## Technologies Used

- HTML5
- CSS3 (custom properties, flexbox, grid)
- JavaScript (vanilla)
